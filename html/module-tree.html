Module Hierarchy

  • cosmiclite: Script qui reduit la taille de la database cosmic en supprimant tout les doublons par rapport aux identifiants cosmic.
  • globalinformations: Script creant un fichier resume et qualite pour chaque echantillon du run.
  • hotspot: TODO : Commentaires preciser HSnm et HSm
  • main_varan: Script principal du pipeline qui traite le fichier .vcf de chaque patients d'un run afin d'obtenir un compte rendu de mutations.
  • refseqtoensembl: Script qui idEnsembleFind les equivalences entre identifiants RefSeq et identifiants Ensembl puis compare les identifiants Ensembl à ceux de Cosmic afin d'obtenir diverses informations.
  • separationvariants: Script pour separer une ligne du fichier VCF composee de plusieurs transcripts en plusieurs lignes: une ligne par transcript.
  • update: Script qui verifie sur le web les dernieres versions des bases de donnees.
  • varan: Script principal du pipeline qui traite le fichier .vcf de chaque patients d'un run afin d'obtenir un compte rendu de mutations.
  • variantfilter: Script qui filtre les variants trouves lors du sequençage.
